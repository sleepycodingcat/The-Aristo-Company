Orange Juice BTS

This film was always an ambitious endevaor. Cloning, masking, a complicated storyline, location changes, easy to get wrong emotions...all very risky. But the thought was, if I don't experiment now, when will I ever? It doesn't seem like a smart move to do these things at IB where failing is not an option - this might be the last time in a while where I'll be able to do something crazy like this.

Although it seems like a fantastical sci fi concept, this film was actually quite personal to me, and I'm glad to have made it.
